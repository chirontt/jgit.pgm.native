Notes on httpclient-4.5.13-native-image-fix.jar
-----------------------------------------------

httpclient-4.5.13-native-image-fix.jar is a recompile of the Apache HttpClient 4.5.13
with a fix: the abstract class org.apache.http.impl.auth.AuthSchemeBase now implements
the Serializable interface, i.e.

public abstract class AuthSchemeBase implements ContextAwareAuthScheme, Serializable {

    private static final long serialVersionUID = 2041257376614171233L;

    ...
}

The Serializable interface is needed to make the AuthSchemeBase class work in native image
generated by the GraalVM native-image tool.
